# 2048: Bomb

클래식 2048 게임에 폭탄 타일을 추가한 변형 버전입니다. 순수 HTML/CSS/JavaScript로 만들어졌으며 별도의 빌드 과정이나 의존성이 없습니다.

## 실행 방법

`index.html`을 브라우저로 열면 바로 플레이할 수 있습니다.

## 규칙

- 방향키(`↑ ↓ ← →`) 또는 `W`/`A`/`S`/`D`, 모바일에서는 스와이프로 타일을 이동합니다.
- 같은 숫자 타일끼리 합쳐지면 값이 두 배가 되고 점수를 얻습니다.
- 보드에 가끔 폭탄 타일(💣)이 등장합니다. 폭탄 타일을 숫자 타일에 합치면 **보드에서 가장 큰 숫자 타일**이 터지면서 보너스 점수를 얻습니다.
- 더 이상 이동하거나 합칠 수 없으면 게임이 종료됩니다.
- 최고 점수는 브라우저의 `localStorage`에 저장됩니다.

## 파일 구성

- `index.html` — 페이지 구조
- `style.css` — 스타일
- `script.js` — 게임 로직
